# Sika Checkout Showcase

Live examples demonstrating different ways to integrate Sika checkout.

**[showcase.withsika.com](https://showcase.withsika.com)**

## Examples

| Example | Description |
|---------|-------------|
| **Redirect** | Classic checkout flow - redirect to hosted payment page |
| **Modal** | Open checkout in a modal overlay |
| **Inline** | Embed checkout directly into your page |
| **Popup** | Open checkout in a popup window |
| **Buy Button** | One-click purchase buttons |
| **Donation** | Custom amount with preset options |
| **Vanilla JS** | Pure HTML/JS without frameworks |
